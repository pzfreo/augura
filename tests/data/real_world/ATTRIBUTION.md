# Real-world STEP fixtures — attribution

These files are unmodified copies of third-party designs, included as
regression fixtures so augura's BREP pipeline is exercised against
production CAD (multi-body assemblies, real fillets/chamfers, vendor
STEP writers) rather than only our own parametric fixtures.

- `framework_expansion_card_selftapping.stp`
  (upstream: `Mechanical/Printable/3D/ExpansionCard_SelfTapping.stp`)
